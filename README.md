# Simple-Crypto-Dashboard
A robust and modern cryptocurrency analysis dashboard built with Python (Flask) and JavaScript. This application provides real-time market data, interactive charts, and advanced features like dark mode and caching mechanisms to optimize performance.
#Features
	•	Real-time Market Data: Fetches live data for top cryptocurrencies using the CoinGecko API.
	•	Interactive Charts: Dynamic price visualization with customizable time ranges (24h, 7d, 30d, 1y) using Chart.js.
	•	Advanced Caching: Implements an intelligent caching system to minimize API usage and prevent rate limits.
	•	Modern UI/UX: Fully responsive design with Tailwind CSS, featuring a seamless Dark Mode toggle.
	•	Robust Backend: Built with Flask using Object-Oriented Programming (OOP) principles for scalability and maintainability.
	•	Resilience: Includes fallback mechanisms to ensure the dashboard remains functional even during API outages.

 #Technologies Used
	•	Backend: Python 3.x, Flask, Requests
	•	Frontend: HTML5, Jinja2, Tailwind CSS, Chart.js, Lucide Icons
	•	API: CoinGecko Public API (v3)
